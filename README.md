
# LArIdTranslator

## Quick Start

```bash
# Clone the repository
git clone https://github.com/huachengcai92/LArIdTranslator2.git

# Go inside the directory
cd LArIdTranslator2

# Install dependencies
npm install

# Start development server
npm run start

# Check correctness of `js`, `ts` and `tsx` files
npm run lint

# Build for production into build/ folder
npm run build
```
